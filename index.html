Hello from Apache inside Docker

Este repositório tem apenas dois arquivos: docker-compose.yml e index.html.

Como rodar

  1. No terminal, execute: docker compose up -d
  2. Acesse: http://localhost:8080
  3. Para parar: docker compose down

Como funciona

O arquivo docker-compose.yml usa a imagem httpd:2.4 e monta a pasta do repositório na raiz pública do Apache (/usr/local/apache2/htdocs).
